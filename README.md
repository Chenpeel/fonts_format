# fonts_format

Standardize fonts in Office files — Chinese → SimSun, English → Times New Roman, Japanese → MS Mincho.

Supports `.pptx` `.docx` `.xlsx`.

## Quick Start

**Web**

```bash
docker compose up --build -d
# open http://localhost:9090
```

## Font Rules

| Character | Font |
|-----------|------|
| Chinese | 宋体 (SimSun) |
| English / Numbers | Times New Roman |
| Japanese | MS Mincho |
